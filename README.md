# Simple-Calculator
## Description
This is a simple GUI calculator built using Python and Tkinter. It performs basic arithmetic operations such as addition, subtraction, multiplication, and division.

## Features
1. Interactive button-based interface
2. Real-time display of user input
3.  Handles errors (e.g., division by zero, invalid expressions)
4. Simple and user friendly design

## Technologies Used
1. Python
2. Tkinter

## How to Run
1. Download or clone the repository
2. Run the file:
   Simple_calculator.py
